# BSC Surge

这是一个基于 Hardhat 的 BSC (Binance Smart Chain) 智能合约开发项目。

## 安装依赖

```bash
npm install
```

## 编译合约

```bash
npm run compile
```

## 运行测试

```bash
npm run test
```

## 启动本地节点

```bash
npm run node
```

## 部署合约

```bash
npm run deploy
```

## 网络配置

项目已配置以下网络：
- `hardhat`: 本地开发网络
- `bsc`: BSC 主网
- `bscTestnet`: BSC 测试网

在 `hardhat.config.js` 中配置你的私钥（建议使用环境变量）。
